Configuration file example:

#Custom comment

#the syntax is :

[var name][space][value(s)];

# #[var name] : Name of the var, without any whitespaces into #[space] : must be a ' ' or a '' #[value(s)] : the value(s) field #The ';' is very important !

#Examples: MyVar 10; MySentense Hello world !; MyArray 10 56 38 52 46;

#A value can contain any character excepting ';'

It can even contain '' !
